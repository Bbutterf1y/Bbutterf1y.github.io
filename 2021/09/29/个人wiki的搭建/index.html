安装

安装python和pip

mkdocs支持pyhton3.5, 3.6, 3.7, 3.8（请参考官方文档，支持版本随时在更新） 附：python下载地址

pip一般在python安装时可以选择自动安装，如需更新可使用 pip install --upgrade pip

如果需要安装请使用 python get-pip.py

安装mkdocs

使用pip安装：pip install mkdocs

安装完成后可以使用 mkdocs --version 查看安装是否正确

入门

创建项目

mkdocs new 项目名称（默认创建在命令行所在目录下）

cd 项目名称 进入项目文件夹

项目文件夹由两个文件组成，一个名为mkdocs.yml的配置文件，以及一个名为docs的文件夹，这个文件夹包含着文档源文件。初始情况docs文件夹只包含一个名为index.md的文档页面。

启动项目

确保与mkdocs.yml配置文件位于同一目录中，然后通过运行mkdocs serve命令启动服务器

在浏览器打开http://127.0.0.1:8080/可以看到初始界面
